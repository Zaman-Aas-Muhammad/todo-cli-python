# SEO & Python Task Manager (CLI)

A command-line based Todo application built in Python
that helps developers and SEO professionals manage
and track their daily tasks efficiently.

## Features
- Add tasks with categories (Python / SEO / General)
- View all tasks in organized format
- Remove completed tasks
- SEO & Python task statistics dashboard
- Smart task detection (auto-identifies SEO/Python related tasks)

## Technologies Used
- Python 3.x
- OOP Concepts
- File Handling

## How to Run

pip install -r requirements.txt
python todo.py

## Usage
1 - Add new task
2 - View all tasks
3 - Remove a task
4 - Show SEO stats
5 - Exit
